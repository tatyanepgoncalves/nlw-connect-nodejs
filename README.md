# Sistema de Indicação de Referral com Ranking

## 🔥 Introdução
Sistema de indicação de referral com ranking feito durante o NLW Connect da Rocketseat.

### Funcionalidades
- Criação de inscrição
- Acesso ao link 
- Ranking de quantidade de inscrição com o link

## ⚙️ Pré-requisitos
- Node JS
- Docker Compose
- Biome
